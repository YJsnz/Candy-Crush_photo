# 糖果消消乐 (Candy Crush)

一个基于浏览器的糖果消消乐游戏，使用 HTML/CSS/JavaScript + jQuery 构建。

## 游戏介绍

这是一个经典的三消类游戏，玩家需要通过交换相邻的糖果，使相同颜色的糖果形成3个或以上的连线，从而消除它们并获得分数。游戏具有简洁的界面和流畅的操作体验，适合所有年龄段的玩家。

## 如何开始游戏

1. 打开 `index.html` 文件（推荐使用 Chrome 浏览器）
2. 游戏会自动加载并生成初始糖果布局
3. 开始你的糖果消除之旅！

## 游戏操作

### 基本操作

1. 在输入框中输入单元格坐标（例如：`a1`、`B3`）— 列使用 `a–h` 字母表示，行使用 `1–8` 数字表示
2. 启用的箭头按钮（↑↓←→）显示有效的移动方向 — 点击按钮交换糖果
3. 点击 **Crush Once** 按钮消除3个或以上相同颜色的糖果组，500毫秒后会自动补充新糖果
4. 点击 **New Game** 按钮重置游戏

### 键盘快捷键

| 按键 | 操作 |
|------|------|
| `n` | 新游戏 |
| `Space`（空格） | 消除一次 |

### 输入提示

- 不区分大小写：`a1` 和 `A1` 都可以正常工作
- 输入单个字母（例如：`a`）可以查看该列中所有有效的坐标

## 游戏规则

1. 只能交换相邻的糖果（上下左右四个方向）
2. 只有当交换后能形成3个或以上相同颜色的连线时，交换才会生效
3. 消除糖果后，上方的糖果会下落填补空位，同时顶部会生成新的糖果
4. 如果一次消除后形成新的3个或以上相同颜色的连线，会自动继续消除

## 文件结构

```
candycrush-skeleton/
├── index.html        # 主游戏文件
├── css/
│   ├── mainLayout.css   # 游戏布局样式
│   └── bootstrap.css    # Bootstrap 框架样式
├── js/
│   ├── candy.js      # 糖果模型
│   ├── board.js      # 游戏板模型和事件处理
│   ├── rules.js      # 游戏规则逻辑
│   ├── jquery.min.js     # jQuery 库
│   └── seedrandom.min.js # 随机数生成库
└── graphics/         # 糖果图片资源
```

## 技术实现

- 使用 HTML5 构建游戏界面
- 使用 CSS3 实现响应式布局和动画效果
- 使用 JavaScript 实现游戏逻辑
- 使用 jQuery 简化 DOM 操作和事件处理
- 使用 seedrandom 库确保随机数生成的一致性

## 开发说明

如果您想修改或扩展游戏功能，可以按照以下步骤进行：

1. 修改 `js/candy.js` 文件可以调整糖果的属性和行为
2. 修改 `js/board.js` 文件可以调整游戏板的大小和交互逻辑
3. 修改 `js/rules.js` 文件可以调整游戏规则和 scoring 系统
4. 修改 `css/mainLayout.css` 文件可以调整游戏界面的样式
5. 在 `graphics/` 目录中添加或替换糖果图片可以更改游戏视觉效果

## 浏览器兼容性

- 推荐使用 Chrome 浏览器以获得最佳体验
- 支持 Firefox、Safari 等现代浏览器
- 不支持 IE 11 及以下版本
